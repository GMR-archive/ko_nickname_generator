# 닉네임 생성기 🌸✨

> **"간단하게 나만의 닉네임을 만들어 보세요!"**

## 📖 소개

**닉네임 생성기**는 클릭 한번에 닉네임을 손쉽게 생성할 수 있습니다.

2글자부터 4글자까지 다양한 길이의 아름다운 닉네임을 클릭 한 번으로 만들어 보세요!

### 💡 주요 기능
- **다양한 단어 조합**: 2글자, 3글자, 4글자 중 선택 가능.
- **랜덤 생성**: 클릭 한 번에 랜덤으로 닉네임이 생성.
- **감성적인 UI**: 단순하고 아름다운 디자인으로 사용 편의성을 극대화.
